### String Basics

---

- str.length : gives length like an array of string.

- str.trim(): removes all whitespaces from string. Can also check if string is empty or not. `emptyStr.trim() === ''`

- ASCII CODE : 
    -   0 to 9 : 48 to 57
    -   a to z : 97 to 122
    -   A to Z : 65 to 90
